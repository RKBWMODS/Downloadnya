INFORMASI

Anda harus menginstall depedensi yang ada di bawah ini agar script berjalan dengan baik.

$ pkg update -y && pkg upgrade -y
$ pkg install python -y
$ pkg install clang -y
$ pkg install git -y
$ pkg install tinyproxy -y
$ termux-setup-storage
$ pip install timedelta
$ pip install pycryptodome
$ pip install requests
$ pip install fake-useragent
$ pip install rich

Ketikan seperti ini di termux

$ nano spam.py

Jika halaman kosong atau hitam silahkan tempel scriptnya ke kotakan hitam, Pastikan semua code di salin penuh agar tidak error.

Ikuti seperti ini agar tersimpan

CTRL, X, Y, ENTER

Runing dengan mengetikan seperti ini.

$ python3 spam.py
